# PM Copilot Codex Shim

This file exists for Codex, where `AGENTS.md` is a repository instruction file.

For PM Copilot behavior, read and follow:

```text
PM_COPILOT.md
```

Do not treat this file as the cross-platform canonical entry. The canonical entry is `PM_COPILOT.md`.

If this repository is nested inside another software project, Codex may also load the host project's own `AGENTS.md` depending on the working directory. In that case, the host project should include a small adapter instruction that explicitly delegates product-manager tasks to `pm-copilot/PM_COPILOT.md`.
