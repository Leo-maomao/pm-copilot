# Clarifying Questions

## Must Answer Before Launch

1. Is offline access available to all users or premium users only?
2. Which content types are eligible or restricted for offline access?
3. What cache size and expiration rules should apply?
4. Is saved content synced across devices in v1?
5. What app versions and operating systems are in scope?

## Can Proceed With Assumptions for Draft

1. Save is available to logged-in users.
2. Saved tab is the primary return path.
3. Offline access is limited to eligible cached content.
4. Restricted content shows an online-only state.
5. The first version does not include folders or a full download manager.

## Can Decide Later

1. Folder or collection organization.
2. Cross-device conflict handling.
3. Offline content expiration messaging.
4. Premium upsell treatment for offline access.
