# Final Package Workflow

## Inputs

- PRD
- Metrics tree
- Tracking plan
- User flow
- HTML prototype
- Review checklist
- Assumptions and open questions

## Packaging Steps

1. Verify each artifact exists.
2. Verify each artifact follows its contract.
3. Create an artifact index with file names and purposes.
4. Summarize the product decision, scope, metrics, prototype platform, and review status.
5. List assumptions, open questions, and required human confirmations.
6. Provide a recommended stakeholder review agenda.

## Final Package Sections

- Executive summary
- Artifact index
- Key decisions
- Metrics and tracking summary
- Prototype summary
- Risks and open questions
- Review agenda
- Next actions
